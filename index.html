AI Data Tool · Python

AutoInsight — Conversational AI Data Analyst

A tool that lets anyone ask a dataset questions in plain English and get back an answer, a chart, and a table. It interprets the question, runs the analysis, and double-checks its own calculations before showing a result. It works without any paid API keys thanks to a deterministic question router, with an optional language-model layer for nicer wording.

Data Engineering · Python

NYC Citi Bike 2023 — Ridership Analysis

A streaming Python pipeline that processed all 35.1 million Citi Bike trips of 2023 — 6.8 GB of nested ZIP/CSV data — entirely in memory, without writing the raw files to disk. The result is a single, self-contained dashboard showing commuter patterns, a strong summer demand peak, and the shift toward electric bikes over the year.

PythonStreaming ETL Data vizHTML / JS
Live dashboard ↗ GitHub ↗
Why it matters Demand planning: the kind of usage-pattern analysis a transit or logistics team would use to plan supply — handled efficiently on a dataset large enough to trip up many tools.
